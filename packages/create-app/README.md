## @okiss/create-app

`OMS` 的项目模板.

```shell
npm init @okiss/app
# or
yarn create @okiss/app
# or
pnpm create @okiss/app

# npm 6.x
npm init @okiss/app m-app --template ts 

# npm 7+, extra double-dash is needed:
npm init @okiss/app m-app -- --template ts

# yarn
yarn create @okiss/app m-app --template js
```

template 的可选项有: `js`, `ts`
