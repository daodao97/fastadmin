# 快速开始

## 前置准备

请预先配置好本地的 `node >= v16.11.0`, `golang >= 1.14` 环境.


## 前端

```shell
pnpm create @okiss/app
```

## 后端

```shell
git clone git@github.com:daodao97/goms.git
```

## 数据库导入

## 启动 

```shell
# 前端
pnpm dev

# 后端
go run .
```


# 使用 oms 命令

## 安装辅助命令

```sh
go get github.com/daodao97/oms-cli
```

## 创建项目
```shell
oms create myapp
```

## 无前端化启动
```shell
oms dev
```
